Adult Income Prediction – Capstone Project

By: Jyothipriya Ramavath | Master’s in Computer Information Systems, Christian Brothers University

Project Overview

This capstone project predicts whether an individual earns more than $50K annually using the UCI Adult Census dataset. It demonstrates a complete end-to-end machine learning pipeline: data cleaning, exploratory data analysis, model development, evaluation, and deployment via FastAPI.

Goal: Build and deploy a production-style ML model that can serve real-time predictions through an API.

Tech Stack & Tools

Python Pandas Matplotlib Scikit-Learn FastAPI Joblib Postman VS Code GitHub

Pipeline Steps

  1. Data Preparation: Loaded raw adult.data, applied official UCI column names, handled missing values, and encoded the target (<=50K vs >50K).
  2. Exploratory Data Analysis: Visualized age, hours worked, income distribution, and relationships between income and education, sex, and marital status.
  3. Modeling: Built both Logistic Regression (baseline) and Random Forest models using a unified preprocessing pipeline (StandardScaler + OneHotEncoder).
  4. Evaluation: Compared models using Accuracy and F1 Score; Random Forest achieved the best performance.
  5. Deployment: Exposed the trained model via a FastAPI endpoint (/predict) and validated it using Postman with JSON payloads.

Key Learnings

  • How to build a full ML workflow from raw data to deployed API.
  • Practical experience with feature engineering, encoding, and model selection.
  • Designing REST APIs with FastAPI and validating with Pydantic models.
  • Testing ML services using Postman and structuring a project for GitHub.
